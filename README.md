# Optimizing Size Recommendations to Reduce Returns at Myntra

A data-driven product strategy project analyzing fashion e-commerce sizing challenges and proposing scalable solutions to reduce return rates.

## ðŸ“Š Key Findings

### 1. Brand Inconsistency
- 33% of brands show >15% size variation (Coefficient of Variation)
- Puma (20.35%) and Adidas (18.77%) show highest inconsistency
- Highlander (5.89%) is the most consistent brand

### 2. Customer Pain Points
- ~24% of customers mention sizing issues in reviews
- Only ~76% report accurate sizing
- Common complaints include misleading size guides and inconsistent fits

### 3. Market Gap
- Amazon is the only platform with structured fit feedback
- No platform offers cross-brand size comparison
- Clear opportunity for Myntra to differentiate on sizing confidence

- Runs Small
